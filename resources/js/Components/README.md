# Vue Components

This directory contains reusable Vue.js components.

## Suggested Structure

```
Components/
├── Common/           # Shared/generic components
│   ├── Button.vue
│   ├── Modal.vue
│   ├── Toast.vue
│   └── Spinner.vue
├── Dashboard/        # Dashboard-specific components
│   ├── ItemCard.vue
│   ├── QuickBuySection.vue
│   ├── ShoppingList.vue
│   ├── InventoryList.vue
│   └── ActivityFeed.vue
├── Forms/            # Form components
│   ├── SmartInput.vue
│   ├── CategorySelect.vue
│   └── RecurringPicker.vue
└── Layout/           # Layout components
    ├── AppLayout.vue
    ├── NavBar.vue
    └── Sidebar.vue
```
